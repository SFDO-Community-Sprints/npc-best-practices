# Migration Guidance

## Introduction

Considering a migration to Agentforce for Nonprofit (AFNP), formerly known as Salesforce Nonprofit Cloud (NPC)? This document is meant to be an open source resource for any org wanting pre-migration guidance, created by the community, for the community. 

No two orgs are the same, so please use this as a *guide*. Use what’s applicable and helpful for your organization and leave the rest!
